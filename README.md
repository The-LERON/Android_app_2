# Happy Birthday — Поздравительная открытка

## Описание
Приложение "Happy Birthday" — поздравительная открытка, созданная с использованием Jetpack Compose.

## Технологии
- **Jetpack Compose** — современный toolkit для создания UI на Android
- **Kotlin** — основной язык разработки
- Минимальная версия SDK: **API 24 (Nougat)**

## Особенности
- Отображение поздравительного текста по центру экрана
- Подпись отправителя с выравниванием по правому краю
- Фоновое изображение
- Использованы Composable-функции: `Column`, `Row`, `Box`, `Text`, `Image`

## Структура
- `MainActivity.kt` — основной файл с UI
- `GreetingText()` — Composable-функция для отображения поздравления и подписи
- `GreetingImage()` — Composable-функция для отображения фонового изображения
